# comic-event-circle-lists

同人誌即売会の配置アーカイブ


## 正規化

サークル名・著者名・サークルスペースはUnicode NFKCで正規化されています。詳しくは[Unicode正規化#正規化形式 - Wikipedia](https://ja.wikipedia.org/wiki/Unicode%E6%AD%A3%E8%A6%8F%E5%8C%96#%E6%AD%A3%E8%A6%8F%E5%8C%96%E5%BD%A2%E5%BC%8F)などを参照してください。

ざっくりいうと

* 全角アルファベットや数字は半角に
* 半角カナは全角に
* 丸数字（①など）は半角数字に

変換されます。
